# Enemy Within

Campaign repository for a Warhammer Fantasy Roleplay 4th Edition playthrough of The Enemy Within.

## Canonical Files

- `character_sheet.md` — current character mechanics and state
- `campaign_bible.md` — player-known campaign continuity
- `gm_journal.md` — private GM notes and adventure state
- `session_log.md` — campaign session index
- `sessions/` — individual session records
